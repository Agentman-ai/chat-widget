Implementation Code

Here's how to embed the chat widget in an iframe:

Parent Page (this page):

<iframe src="chat-iframe-content.html" width="100%" height="600"></iframe>

Iframe Content Page:

<!DOCTYPE html> <html> <head> <!-- Load widget from CDN --> <script src="https://unpkg.com/@agentman/chat-widget@latest/dist/index.js"></script> <style> body { margin: 0; padding: 0; } #chat-widget { height: 100vh; } </style> </head> <body> <div id="chat-widget"></div> <script> new ChatWidget({ agentToken: "eJyrVipJzUvMK4nPTFGyMjHTUUosKMjJTE4syczPQ4ilp0JVGBqa1wIAxoEQ6Q.Z32csg.xbR3NqyuNlDpt-Dafp1O2GpKw5U", apiUrl: 'https://studio-api.agentman.ai', containerId: 'chat-widget', variant: 'inline', // Best for iframes title: 'Iframe Chat Demo' }); </script> </body> </html>

Benefits of Iframe Integration

  • Isolation: The widget runs in a separate context, preventing CSS/JS conflicts
  • Security: Enhanced security through iframe sandboxing
  • Easy Integration: No need to modify existing page styles or scripts
  • CDN Loading: Widget loads directly from CDN, no local files needed
  • Cross-Domain: Can be hosted on different domains

Configuration Tips

  • Use variant: 'inline' for best iframe experience
  • Set appropriate iframe dimensions based on your layout
  • Consider using sandbox attribute for additional security
  • Test responsiveness on different screen sizes
